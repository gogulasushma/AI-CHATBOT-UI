
# 🤖 AI Chatbot

An AI-powered chatbot built using Python, Flask, HTML, CSS, and JavaScript. The chatbot provides real-time responses using an AI API through a clean and responsive web interface.

## 🚀 Features

- Interactive chatbot interface
- AI-generated responses
- Responsive and user-friendly design
- Fast and simple web application
- Built using Flask backend

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- AI API

## 📁 Project Structure

```
AI-Chatbot/

│── templates/
│     └── index.html
│── static/

```

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/your-username/AI-Chatbot.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Create a `.env` file and add your API key.

4. Run the application

```
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```

## 📸 Output

The chatbot accepts user questions and generates AI-powered responses in real time through a simple web interface.
